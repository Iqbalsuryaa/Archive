Modul 3 – Linked List

  1. Buatlah program dari 4 node berikut dengan kondisi awal linked list kosong:

  2. Tambahkan node baru dengan data 500 dari belakang. Tambahkan node baru dengan data 50 dari depan. Tambahkan node dengan data 250 setelah node 200.

  3. Hapus node depan. Selanjutnya hapus node belakang. Selanjutnya hapus node yang memiliki data 300.

  4. Akses semua data dari seluruh node tersebut dari node yg paling depan ke belakang.

class Node:
    def __init__(self, data):
        self.data = data
        self.next = None

class LinkedList:
    def __init__(self):
        self.head = None

    def PenambahanData_DariBelakang(self, data):
        node_baru = Node(data)
        if self.head is None:
            self.head = node_baru
            return
        last_node = self.head
        while last_node.next:
            last_node = last_node.next
        last_node.next = node_baru

    def PenambahanData_DariDepan(self, data):
        node_baru = Node(data)
        node_baru.next = self.head
        self.head = node_baru

    def PenambahanData_SetelahNode(self, data_setelah, data):
        node_baru = Node(data)
        node = self.head
        while node:
            if node.data == data_setelah:
                node_baru.next = node.next
                node.next = node_baru
                return
            node = node.next
        print(f"{data_setelah} tidak ditemukan dalam linked list")

    def HapusNode_Depan(self):
        if self.head is None:
            print("Linked List kosong.")
            return
        self.head = self.head.next

    def HapusNode_Belakang(self):
        if self.head is None:
            print("Linked List kosong.")
            return
        if self.head.next is None:
            self.head = None
            return
        node = self.head
        while node.next.next:
            node = node.next
        node.next = None

    def penghapusan_data(self, data):
        if self.head is None:
            print("Linked List kosong.")
            return
        if self.head.data == data:
            self.head = self.head.next
            return
        node = self.head
        while node.next:
            if node.next.data == data:
                node.next = node.next.next
                return
            node = node.next
        print(f"{data} tidak ditemukan dalam linked list")

    def pengakses_semua(self):
        if self.head is None:
            print("Linked List kosong.")
            return
        node = self.head
        while node:
            print(node.data, end=" ")
            node = node.next
        print()


# Inisialisasi linked list kosong
linked_list = LinkedList()

# Tambahkan node awal dengan data 100, 200, 300, 400
linked_list.PenambahanData_DariBelakang(100)
linked_list.PenambahanData_DariBelakang(200)
linked_list.PenambahanData_DariBelakang(300)
linked_list.PenambahanData_DariBelakang(400)
print("Linked List awal:")
linked_list.pengakses_semua()

# Tambahkan node baru dengan data 500 dari belakang, 50 dari depan, 250 setelah node 200
linked_list.PenambahanData_DariBelakang(500)
print("Linked List setelah Penambahan Data Dari Belakang(500):")
linked_list.pengakses_semua()

linked_list.PenambahanData_DariDepan(50)
print("Linked List setelah PenambahanData Dari Depan50):")
linked_list.pengakses_semua()

linked_list.PenambahanData_SetelahNode(200, 250)
print("Linked List setelah Penambahan Data Setelah Node(200, 250):")
linked_list.pengakses_semua()

# Hapus node depan, belakang, dan yang memiliki data 300
linked_list.HapusNode_Depan()
linked_list.HapusNode_Belakang()
linked_list.penghapusan_data(300)
print("Linked List setelah hapus depan, belakang dan 300 :")
linked_list.pengakses_semua()

Penejelsan Codingan Diatas :

Beberapa penjelasan mengenai method yang ada pada class LinkedList diatas adalah:

  1. PenambahanData_DariBelakang(data) : Method ini digunakan untuk menambahkan data baru pada akhir linked list.

  2. PenambahanData_DariDepan(data) : Method ini digunakan untuk menambahkan data baru pada awal linked list.

  3. PenambahanData_SetelahNode(data_setelah, data) : Method ini digunakan untuk menambahkan data baru setelah node tertentu dalam linked list.

  4. HapusNode_Depan() : Method ini digunakan untuk menghapus node pertama dalam linked list.

  5. HapusNode_Belakang() : Method ini digunakan untuk menghapus node terakhir dalam linked list.

  6. penghapusan_data(data) : Method ini digunakan untuk menghapus node yang memiliki data tertentu dalam linked list.

  7. pengakses_semua() : Method ini digunakan untuk mencetak semua data yang terdapat pada linked list.

Kelas pertama adalah Node, yang berfungsi sebagai blueprint untuk membuat objek node baru pada Linked List. Setiap node memiliki atribut data yang menyimpan nilai data dan atribut next yang menyimpan pointer ke node selanjutnya.

Kelas kedua adalah LinkedList, yang berfungsi untuk mengelola seluruh node pada Linked List. Pada inisialisasi, objek LinkedList dibuat dengan atribut head yang menunjukkan node pertama pada Linked List.

Metode PenambahanData_DariBelakang digunakan untuk menambahkan node baru dengan data pada akhir Linked List. Pada metode ini, jika Linked List masih kosong (head == None), maka node baru langsung dijadikan head. Jika tidak, maka pointer next pada node terakhir di-iterate hingga pointer tersebut menunjuk ke node baru.

Metode PenambahanData_DariDepan digunakan untuk menambahkan node baru dengan data pada awal Linked List. Pada metode ini, node baru dijadikan head, dan pointer next-nya diarahkan ke node yang sebelumnya menjadi head.

Metode PenambahanData_SetelahNode digunakan untuk menambahkan node baru dengan data setelah node tertentu pada Linked List. Pada metode ini, Linked List di-iterate hingga menemukan node dengan data yang sesuai. Jika ditemukan, node baru ditambahkan setelah node tersebut dengan memodifikasi pointer next pada node sebelumnya dan node baru.

Metode HapusNode_Depan digunakan untuk menghapus node pertama pada Linked List. Pada metode ini, jika Linked List masih kosong, maka akan muncul pesan error. Jika tidak, head diubah menjadi node selanjutnya.

Metode HapusNode_Belakang digunakan untuk menghapus node terakhir pada Linked List. Pada metode ini, jika Linked List masih kosong atau hanya memiliki satu node, maka akan diubah menjadi Linked List kosong. Jika tidak, pointer next pada node kedua terakhir di-iterate hingga pointer tersebut menunjuk ke None.

Metode penghapusan_data digunakan untuk menghapus node dengan data tertentu pada Linked List. Pada metode ini, Linked List di-iterate hingga menemukan node dengan data yang sesuai. Jika ditemukan, pointer next pada node sebelumnya diubah untuk menunjuk ke node setelahnya.

Metode pengakses_semua digunakan untuk mencetak semua data pada Linked List secara terurut.

Pada program utama, objek Linked List dibuat dan diisi dengan node-node baru dengan menggunakan metode PenambahanData_DariBelakang. Kemudian, dilakukan penambahan node baru dari belakang, depan, dan setelah node tertentu menggunakan metode yang sesuai, serta penghapusan node dari depan, belakang, dan dengan data tertentu. Setelah itu, seluruh data pada Linked List dicetak menggunakan metode pengakses_semua.
